# Demo application with NestJs for the server side and Stencil for the Client

## Install Server

```sh
  cd server 
  npm install
```

## Install Client Side

```sh
  cd client
  npm install
```

## Run the server
Create a run .env file like the file .env.demo
You'll need to connect to your local database
Finally execute

```sh
  npm start
```

## Run the client

```sh
  npm start
```
